# 工具与自动化

具备工具调用能力的模型可以使用内置工具与 MCP 扩展来补充上下文或直接执行操作。可在完全信任模型与端点时开启“跳过工具确认”实现自动执行。

![工具配置界面](../../../res/screenshots/imgs/ai-assistant-tool-configuration-interface.png)

## 内置工具

| 工具组 | 功能 |
| --- | --- |
| 日历 | 读取忙闲、建议时间、创建/更新事件。 |
| 网页浏览/抓取 | 搜索或抓取网页并返回结构化内容，详见 [网络搜索](./web_search.md)。 |
| 定位 | 获取当前位置/经纬度/逆地理（iOS/iPadOS）。 |
| URL 元数据 | 展开链接标题、描述与图标。 |
| 记忆 | 存储、回忆、更新或删除长期记忆。 |
| MCP | 连接外部 MCP 服务器扩展工具，详见 [MCP 集成](./mcp_integration.md)。 |

## 开启方式

- 会话内：输入框上方 **工具** 开关（仅在模型支持时显示）；长按可批量开关内置工具与 MCP。**网页浏览** 控制当前对话能否联网搜索。
- 全局：在 **设置 → 工具** 管理内置工具、MCP 服务器、搜索引擎；需要全自动化时，可启用 **跳过工具确认**。

## 运行流程

1. 模型发送函数名与 JSON 参数（含动态 MCP 工具）。
2. 你确认后执行；若启用自动化则直接执行。
3. 结果写入对话并在推理面板实时展示，可继续串联调用。

## 多模态与附件

- 支持文本/Markdown、PDF、常见图像、音频（自动转码 WAV）；可拖拽、粘贴或点击回形针导入。
- 媒体仅在模型声明相应能力时回传；超大附件可能延长同步时间。
