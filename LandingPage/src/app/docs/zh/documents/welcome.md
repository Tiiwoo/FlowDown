# 欢迎使用浮望

浮望（FlowDown）是一款注重隐私的 AI 工作空间，它无缝整合了本地推理与各类 OpenAI 兼容服务，为你提供流畅、优质的智能体验。

![截图](../../res/screenshots/imgs/flowdown-onboarding-feature-highlights.png)

## 安装指南

推荐从 [App Store](./app_store) 安装正式版以获得最佳体验。高级用户也可以访问 GitHub 获取源码自行编译。

## 关于浮望

浮望致力于在轻量简洁的界面与强大的模型能力之间找到完美平衡。

无论是日常的轻量级问答，还是探索前沿模型的复杂特性，你都可以在本地 MLX、Pollinations、Apple Intelligence 以及自托管端点之间自由切换。

我们深知隐私的重要性。在浮望中，你的数据完全掌握在自己手中——所有会话默认存储在本地，并支持选择性地通过 iCloud 进行安全同步。

### 快速体验

应用内置了来自 [pollinations.ai](https://pollinations.ai/) 的免费云端模型，方便你首次上手体验。应用会自动获取模型列表，并为免费端点标记工具调用、视觉与音频能力。

> **注意**：免费端点的能力与列表刷新频率可能有限，为了获得更稳定、高效的体验，建议尽快在 **设置 → 模型** 中配置你的私有端点或本地模型。

### 反馈与隐私

如遇问题，在提交反馈前请务必进行脱敏处理，切勿上传服务凭证或个人敏感信息。你可以在截图前涂抹敏感字段。更多关于数据处理的原则，请参阅[隐私政策](./legal/privacy.md)。

## 主要特性

- **沉浸式体验**：精心打磨的界面，提供丝滑流畅的 Markdown 渲染效果。
- **多端同步**：基于 CloudKit，让你的会话、模型配置、记忆与 MCP 服务在 iPhone、iPad 和 Mac 间保持一致。支持手动刷新及同步范围控制。
- **混合模型生态**：原生支持 Apple Intelligence、本地 MLX 模型、Pollinations（自动标注能力），以及任意 OpenAI 兼容 API（支持自定义请求头）。
- **灵活配置与分享**：每个模型均可独立设置温度、上下文长度、请求体及辅助行为。支持通过 `.fdmodel` 文件导入导出，或直接拖拽分享配置。
- **记忆与自动化**：支持主动记忆注入对话与快捷指令。系统会自动生成会话标题和图标，减少重复操作，提升效率。
- **强大的工具链**：集成日历管理、网络搜索/抓取、定位服务、URL 预览及 MCP 工具链，让复杂的任务流程也能实现自动化。
- **快捷指令集成**：深度集成系统快捷指令，支持提问、摘要生成或创建会话链接（iOS 18/macOS 15 支持图像输入），并支持数据导出与日志管理。

## 版本对比

浮望目前提供两个版本：持续更新的 **App Store 正式版** 与已存档的 **社区版**。为了获得 CloudKit 同步、现代化的界面设计以及持续的安全更新，建议迁移至正式版。

| 沉浸式界面               | ✅                 | ❌             |
| 模型级参数配置           | ✅                 | ❌             |
| MLX 本地 & Apple FM 支持 | ✅                 | ❌             |
