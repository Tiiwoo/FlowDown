# 常见问题

本页汇总了常见问题。如果未找到答案，请参考[问题反馈指南](./report_issue.md)或加入 Discord 与我们讨论。

## 购买与试用

**购买前是否提供试用？**

- 提供。请加入 [Discord 社区](https://discord.gg/UHKMRyJcgc) 或前往 GitHub 获取最新的 TestFlight 链接，也可以按照仓库说明自行编译体验。
- 应用内置了免费的推理服务，但不保证可用性，建议尽快配置自备的服务节点。

**后续是否还有限免或优惠？**

- 2025 国庆限免活动已于 2025-10-06 结束。后续优惠信息将通过应用更新日志与社交渠道发布。

> **关于价格**：我们计划明年将浮望改为免费软件。随着更多开发者加入，以及基础功能完善后维护成本降低，我们能以较低成本运营（仅覆盖域名、服务器、文档站和开发者账号等必要费用）。在转为免费之前，浮望将分阶段降价。对话和聊天功能将保持免费，仅对后续新增的个性化功能收取订阅费。具体时间表请查看 [App Store 下载页的价格调整](../app_store.md#价格调整)。

## 同步与数据

**如何开启 iCloud 同步？**

- 登录 iCloud，打开 **设置 → 数据控制 → iCloud 同步**。若遇异常，请在同页使用“完全刷新/重置同步状态”，仍有问题时导出日志。
- 详细操作与注意事项见 [数据备份与重置](../settings/data_backup.md)。

![同步范围配置界面](../../../res/screenshots/imgs/flowdown-data-synchronization-configuration.png)

**是否支持备份或重置？**

- 支持。请参考 [数据备份与重置](../settings/data_backup.md) 获取最新步骤与风险提示。升级或回滚前务必先导出备份。

## 模型与推理

**当前支持哪些模型？**

- Apple Foundation 模型（需设备支持 Apple Intelligence）。
- Pollinations AI，会自动获取最新模型列表。
- 任何符合 OpenAI 接口规范的服务，例如 OpenRouter 等。

**模型参数如何生效？**

- 每个模型档案均可独立设置温度、上下文长度、自定义请求体和辅助行为，修改仅对当前档案有效。
- 沉浸式界面仅影响视觉呈现，不会改变模型输出内容。

## 工具与自动化

**如何快速调整工具？**

- 在输入栏长按即可唤出工具快捷菜单。
- 详细配置位于 **设置 → 工具**，可选择搜索引擎，也可设置执行前是否确认。

**有哪些自动化能力？**

- 内置工具涵盖日程规划、网页搜索/抓取、定位（iOS/iPadOS）、URL 丰富和记忆管理。
- MCP 服务器可扩展远程工具，在 **设置 → 工具 → MCP 服务器** 中可测试连接。
- Apple Shortcuts 提供提问、设置默认模型、总结记录以及（iOS 18/macOS 15）带图提问等功能。

## 连接与合规

**是否提供第三方服务的账号或教程？**

- 不提供。

**支持自建推理服务吗？**

- 支持。只要是兼容 OpenAI 接口的网关均可使用，注意网络连通性与限流配置。

## 故障排查

**持续提示同步失败如何处理？**

- 在 **设置 → 数据控制** 关闭同步，然后点击 **重置同步状态**。
- 重新开启同步，等待状态指示灯变绿。
- 若问题依旧，请导出脱敏日志提供给支持团队。

**诊断日志在哪里？**

- 进入 **设置 → 支持**，导出日志，请移除敏感信息后附在工单或邮件中。
