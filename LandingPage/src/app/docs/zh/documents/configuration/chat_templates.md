# 对话模版

对话模版功能允许您在新会话中快速复用系统提示词和表情头像。模版仅存储名称、表情头像、系统提示词，以及创建会话时是否继承全局系统提示词的设置；它不会记录工具配置、开场消息或模型偏好。

## 创建或导入

- **基于现有会话生成**：在会话菜单中选择「生成对话模版」。浮望将使用当前聊天模型，从最近的对话中提取名称、表情和提示词（需包含至少一条用户消息与助手回复），并保存至「设置 → 模型 → 对话模版」。
- **新建空白模版**：前往「设置 → 模型 → 对话模版」，点击 **＋ → 创建模版**，即可编辑空白模版的提示词、头像和名称。
- **重写已有模版**：在模版编辑页点击左上角的 AI 辅助按钮（火花图标），输入指令，让默认模型协助您优化提示词。
- **导入**：在模版列表选择 **＋ → 从文件导入** 载入 `.fdtemplate` 文件（支持多选）。您也可以直接将 `.fdtemplate` 文件拖拽到列表，或从 Finder/分享面板打开导入。注意：相同 ID 的模版将被替换。

## 编辑与管理

- 在模版编辑页，您可以修改表情头像、名称、提示词，以及设置是否继承全局提示词。点击确认按钮即可保存更改。
- 您可以通过「创建副本」复制模版，通过「导出模版」生成 `.fdtemplate` 文件进行分享，或通过「删除模版」移除不再需要的模版。模版保存在本地，列表支持拖拽排序；右滑或长按菜单也可进行删除操作。

![模版配置与生命周期管理](../../../res/screenshots/imgs/template-configuration-and-lifecycle-management-interface.png)

## 使用模版

- 在新建会话菜单中选择 **＋ → 选择模版**（包含浮动新建按钮）即可启动会话；macOS 菜单栏也提供了“使用模版新建会话”选项。
- 应用模版后，系统会自动设置会话图标与名称，并将提示词写入系统消息。如果您选择了“忽略全局提示词”，系统会先清空默认的系统提示，并附带说明提示词来源。

![新建会话选择模版](../../../res/screenshots/imgs/ai-conversation-management-and-template-selection-interface.png)

## 使用建议

- **精简提示词**：保持提示词简洁明了，以便在不同模型间通用。
- **搭配推理控制**：可与[推理配置](../models/inference_configuration.md)搭配，在需要深入思考的任务中预先分配推理预算。
