# 数据备份与重置

了解如何安全地导出、备份数据，或在必要时重置应用状态。

## 导出设置与备份

- 使用 **设置 → 数据 → 导出设置** 备份应用偏好与模型配置（不含会话）。
- 需要完整聊天记录与云端模型配置时，使用 **导出工作区 / 导出数据库**。

![设置导出与备份菜单](../../../res/screenshots/imgs/flowdown-settings-data-export-and-backup-ui.png)

## 重置与风险提示

- **重置应用** 属于不可逆操作，所有数据会被清空。执行前请务必先 **导出备份**。
- 若为同步异常，优先尝试 **重置同步状态**，再考虑完整重置。

![重置确认弹窗](../../../res/screenshots/imgs/data-management-and-editing-reset-confirmation.png)

## 建议

- 在更新 TestFlight 或更换设备前，建议先导出完整工作区备份。分享备份文件时，请注意 **移除包含推理服务凭证的敏感配置**。
- 重置后重新检查 iCloud 开关与同步范围再开启同步。
