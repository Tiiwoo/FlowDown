# 通用与外观设置

打造您专属的浮望体验。您可以在此调整 **外观主题**、**品牌名称**，或优化 **编辑器** 与 **模型展示** 的交互逻辑。

## 外观与主题

- 在 **浅色 / 深色 / 跟随系统** 间切换，可调整 Markdown/聊天内容字号。
- 应用会自动适配 iOS/iPadOS/macOS 的动态字体与辅助功能字号设置。

![主题与品牌选项](../../../res/screenshots/imgs/general-display-and-theme-configuration.png)

## 品牌名自定义

- 可将顶部显示名称改为企业或个人名称；留空则恢复默认 “FlowDown”。
- 适合共享设备或需要品牌露出的场景。

## 编辑与发送偏好

- **发送确认**：避免误触发送（Command/Ctrl + Enter）。
- **粘贴为文件**：大段内容自动转为附件，避免提示词过长。
- **压缩图片**：发送前压缩图片；无论是否压缩都会移除 EXIF。

![编辑与发送偏好](../../../res/screenshots/imgs/configuring-general-chat-and-editor-preferences.png)

## 模型名称展示

- 控制模型标签的显示方式（全名 / 简名 / 隐藏），以减轻输入栏拥挤。

![模型名称显示选项](../../../res/screenshots/imgs/general-settings-and-model-display-configuration.png)

## 快捷外观面板

- 无需深入菜单，可通过快捷面板即时调整主题、字号与品牌名称。
