# Apple 快捷指令

在 iOS、iPadOS 与 macOS 上安装浮望后，您可以在 **快捷指令** App、Spotlight、小组件或 Siri 中调用一组 App Intent 来自动化常见操作。

**注意：** Apple 的快捷指令运行时限制约 30 秒，编排长流程时建议选择轻量模型或降低推理预算。

## 快速开始

1. 打开 **快捷指令**，点击右上角 **＋** 新建快捷指令。
2. 搜索 "FlowDown"，即可看到浮望提供的所有动作。
3. 选择需要的动作，配置模型/消息/文件/音频/候选项等参数，再与日历、提醒事项、文件等系统动作串联。
4. 可将快捷指令添加到桌面或设置自定义的 Siri 口令，便于快速触发。

## 生成与多媒体动作

| 快捷指令                                                     | 功能说明                                               | 关键参数                                                                           |
| ------------------------------------------------------------ | ------------------------------------------------------ | ---------------------------------------------------------------------------------- |
| **生成模型回复**                                             | 发送提示词，可选保存为新会话，并在支持时启用记忆工具。 | `Model`（可选）、`Message`、`Save to Conversation`、`Enable Memory Tools`          |
| **生成模型回复（可携带图片）** _(iOS/iPadOS 18+，macOS 15+)_ | 可附带图片，需选择支持视觉的模型。                     | `Model`、`Message`、`Image`（可选）、`Save to Conversation`、`Enable Memory Tools` |
| **转录音频** _(iOS/iPadOS 18+，macOS 15+)_                   | 使用支持音频的模型将音频转成文本，可选保存结果。       | `Model`、`Audio`、`Language Hint`（可选）、`Save to Conversation`                  |

## 分类动作

| 快捷指令                                   | 功能说明                                                 | 关键参数                            |
| ------------------------------------------ | -------------------------------------------------------- | ----------------------------------- |
| **分类文本**                               | 基于文本在候选项中选择最合适的标签，不确定时返回第一项。 | `Model`、`Content`、`Candidate A-D` |
| **分类图像** _(iOS/iPadOS 18+，macOS 15+)_ | 基于图片在候选项中选择最合适的标签，需视觉模型。         | `Model`、`Image`、`Candidate A-D`   |

## 写作增强类动作

| 快捷指令            | 输出效果                   |
| ------------------- | -------------------------- |
| **总结文本**        | 将输入内容压缩为简洁段落。 |
| **列表总结文本**    | 生成要点式摘要。           |
| **改进写作 - 专业** | 调整为专业、自信的语气。   |
| **改进写作 - 友好** | 调整为温暖、友好的语气。   |
| **改进写作 - 简洁** | 删减冗余，使内容更精炼。   |

上述写作类动作均支持选择 `Model`，可按需求路由至本地 MLX、Pollinations 或其他云端配置。

## 会话与导航动作

| 快捷指令           | 功能说明                                                                         | 关键参数                                  |
| ------------------ | -------------------------------------------------------------------------------- | ----------------------------------------- |
| **创建对话链接**   | 生成 `flowdown://new/<message>` 深链，配合 "打开 URL" 动作启动新会话。           | `Initial Message`                         |
| **获取最近的对话** | 返回最近一次会话的完整记录（含推理），需要解锁设备。                             | -                                         |
| **搜索会话**       | 按关键词搜索已保存的会话（留空则返回最近记录），并输出格式化摘要，支持 1-50 条。 | `Keyword`、`Result Limit`                 |
| **新建会话**       | 创建空白会话，可选择立即切换过去。                                               | `Switch to Conversation`                  |
| **填充对话消息**   | 向指定会话的编辑器预填文本、图片与音频，可与 “展示/展示并发送” 搭配使用。        | `Conversation`、`Text`、`Images`、`Audio` |
| **展示会话**       | 打开浮望并定位到指定会话，不会发送消息。                                         | `Conversation`                            |
| **展示并发送会话** | 打开浮望并触发发送已预填的消息。                                                 | `Conversation`                            |
| **打开浮望**       | 仅启动应用，不附加额外操作。                                                     | -                                         |

## 模型管理

| 快捷指令         | 功能说明                                                   |
| ---------------- | ---------------------------------------------------------- |
| **设置对话模型** | 设置新建会话的默认模型，也作为其他动作未指定模型时的兜底。 |

## 使用技巧

- 需要留档时开启 **保存到对话**，快捷指令会生成以 **快速回复** 命名的会话并保存附件和推理内容。
- 开启 **启用记忆** 可注入主动记忆上下文，并在模型支持工具时写回记忆。
- 图片/音频类动作需搭配具备视觉或音频能力的模型；若模型不支持，请移除对应附件。
- 组合 **填充对话消息** -> **展示并发送会话** 可自动发送预填内容；如需新会话，可先用 **新建会话**。
- 使用 **创建对话链接** 配合 "打开 URL" 触发；若执行失败，可在快捷指令日志查看缺失模型或空消息等错误。
